# Proposal: Talk Question Board

## Why
Live talks attract a flood of audience questions, but the best questions get buried and speakers lose track of what was answered versus skipped.

## What changes
Build a live audience question board where attendees can submit and vote on questions, while the speaker can mark questions as active, answered, or declined. Add an AI polishing step that rewrites each submitted question into a sharper stage-ready version and tags its topic.
